UC00604: Programar para web, na vertente frontend (cliente-side)

Professor responsável: Miguel Romão

Aluno: Rodrigo Cesar

O presente trabalho teve como objetivo o desenvolvimento de ficheiros em HTML, CSS e JavaScript, visando a compreensão das principais tecnologias utilizadas na construção de páginas web dinâmicas e interativas.

No âmbito da estruturação de conteúdos, foram aplicados conhecimentos de HTML para a organização semântica dos elementos da página, incluindo textos, formulários e componentes visuais.

Relativamente ao design e à apresentação, foram utilizados recursos de CSS para a formatação e personalização de elementos, nomeadamente estilos tipográficos, cores, dimensões, espaçamentos e alinhamentos, assegurando uma interface visualmente organizada e funcional.

Adicionalmente, procedeu-se à implementação de funções em JavaScript, com o propósito de introduzir interatividade e processamento lógico, permitindo a manipulação de dados inseridos pelo utilizador e a atualização dinâmica de conteúdos na página.

Por fim, o trabalho possibilitou compreender a integração entre HTML, CSS e JavaScript, evidenciando a interdependência entre estrutura, apresentação e comportamento no desenvolvimento web.
